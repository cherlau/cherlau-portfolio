## 🎯 Sobre o Projeto
Este é o meu cartão de visitas no mundo digital. O objetivo principal foi criar um site leve, sem frameworks pesados, demonstrando domínio em **Vanilla JavaScript** e técnicas modernas de **CSS**.

## ✨ Funcionalidades
- [x] Layout totalmente responsivo.
- [x] Seções de Projetos e Experiência dinâmicas.
- [x] Clareza visual e interface intuitiva para uma navegação fluida.
- [x] Performance.

## 🛠️ Instalação e Uso
Para visualizar o código e fazer alterações locais:
1. Clone este repositório: `git clone git@github.com:cherlau/cherlau-portfolio.git`
2. Abra o `index.html` no seu navegador.
